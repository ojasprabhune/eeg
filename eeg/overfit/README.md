# Overfitting Validation

This module contains simplified versions of datasets and models used to empirically verify the training pipeline through overfitting on very small data samples.

## Subdirectories

### [position_llm/](position_llm/)
Simplified Position LLM components and tokenizers tailored for overfitting tests.

### [data_collection/](data_collection/)
Minimal data loading utilities for overfitting experiments.
